VPN Loop Automator

- Connect to the VPN (NetworkManager via `nmcli`)
- Open a specific website in a Selenium-controlled browser
- Wait 30 seconds
- Close the browser
- Disconnect the VPN
- Wait 5 seconds
- Reconnect the VPN
- Repeat forever

Requirements:

- Linux with NetworkManager (nmcli)
- Python 3.8+
- Selenium
- Firefox + geckodriver installed and accessible in the PATH
